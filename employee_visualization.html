Employee Department Visualization

This page shows the Python code and the resulting department distribution chart.

Python Code

import pandas as pd
import seaborn as sns
import matplotlib.pyplot as plt

# Load the dataset
df = pd.read_csv("employee_data.csv")   # your 100-row dataset

# Frequency count for Sales department
sales_count = (df["department"] == "Sales").sum()
print("Frequency count of Sales department:", sales_count)

# Histogram / Count Plot of all Departments
plt.figure(figsize=(8, 5))
sns.countplot(data=df, x="department", palette="viridis")

plt.title("Distribution of Employees Across Departments", fontsize=14)
plt.xlabel("Department", fontsize=12)
plt.ylabel("Employee Count", fontsize=12)

plt.tight_layout()
plt.show()
